# My React + TypeScript + Vite SPA

This is a SPA application build with React library, Typescript language and Vite bundler.
It is made for learning purposes which uses the API: https://rickandmortyapi.com/.
It's a personal vision on how the "Rick and Morty" episodes and character information could be displayed on the UI.
